## Methodology

This wiki uses the **LLM Wiki** pattern. Instead of using a traditional RAG system that "re-discovers" knowledge from scratch every query, this system relies on an LLM as a *maintainer* that integrates new information into an existing structure of interlinked pages. The result is a *Persistent Compounding Artifact* — a knowledge base whose value grows over time.
